チームC「ChatGPG」	作品名「UniHockey」
作品概要
二人対戦のホッケーゲームです。
能力の違う二体のキャラを選択して戦います。
相手のゴールにホッケーを入れると得点になり、時間内により多く点を取ったほうの勝利になります。

操作方法
キーボード				｜ゲームパッド
WASD		：移動		｜左スティック	：移動
Space		：決定/蹴り	｜Aボタン		：決定/蹴り
				｜Xボタン		：スキル
